## Problem Statement
MTN agents struggle with high volumes of customer complaints, leading to slow resolution times, poor customer experience, increased churn, and revenue leakage.

## Solution
**AI4Telco-MTN-Agent** is an intelligent AI assistant that helps customer service agents:
- Analyze customer complaints in real-time
- Recommend the **Next Best Action**
- Generate professional responses
- Flag high-risk cases (churn, fraud, mass outage)

## Key Features
- Single & Bulk Complaint Analysis
- Next Best Action Recommendation
- Suggested Customer Response
- Risk Scoring & Escalation Logic
- Interactive Insights Dashboard
- MTN & Nigeria-specific context (Pidgin support)

## Tech Stack (Using Hackathon Recommended Tools)
- **Frontend**: Streamlit
- **AI Engine**: Azure OpenAI
- **Hosting**: Streamlit / Render / Railway
- **Data**: Synthetic + Hackathon Datasets
